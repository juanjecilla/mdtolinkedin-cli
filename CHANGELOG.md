## 0.1.0

- Initial release of the `mdtolinkedin` CLI.
- Markdown conversion to LinkedIn-friendly Unicode formatting.
- CLI supports stdin/stdout and file I/O.
- Character limit warnings (default 3000, configurable).
- Optional JSON output for automation.
- Code block handling modes: omit, text, Carbon URL, or rendered images.
